# Calendrier VFC Ligue 3 2026-2027

Ce dépôt publie un fichier `vfc-ligue3.ics` via GitHub Pages.

URL finale attendue :

```text
https://TON-PSEUDO.github.io/vfc-calendrier/vfc-ligue3.ics
```

## Modifier un match

Ouvre `matches.json`, modifie la date ou l’heure, puis clique sur **Commit changes**.
Le fichier `.ics` sera régénéré automatiquement par GitHub Actions.

## Lancer manuellement la publication

Onglet **Actions** > **Publier le calendrier VFC** > **Run workflow**.

## Activation GitHub Pages

Dans le dépôt : **Settings** > **Pages** > **Build and deployment** > **Source** = **GitHub Actions**.
